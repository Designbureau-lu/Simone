# SIMONE

## Core Principles

### 1. The artwork is sacred.

SIMONE never redraws, invents or interprets artwork.
It only maps existing pixels.

### 2. Geometry and appearance are independent.

First we compute where a pixel column goes.
Only afterwards do we decide how it is displayed.

### 3. Understanding is more important than realism.

SIMONE is not a curtain renderer.
It is a design tool for evaluating graphic visibility.
SIMONE is a renderer, not an image filter.
